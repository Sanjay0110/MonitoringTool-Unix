File & Navigation

Command Description
ls List out files and directories
ls –a List out all files and directories including hidden ones
cd (dir) Change working directory to dir
cd or cd ~ Move to user’s home directory
cd .. Move back to the parent directory
cd / Change to root directory
pwd Display the current working directory

System Information

Command Description
cat /proc/cpuinfo Displays CPU information
cat /proc/meminfo Displays Memory information
whoami Displays the username of the logged in user
w List of all logged in users with username, login time, etc
finger (user) Displays username, login, home directory, shell, etc
uptime Tells how long the system is running
df Displays the amount of free disc space on all file systems
free Displays memory and swap space details
uname –a Shows hardware and software information (arch, Kernel, etc.)
man (command_name) Displays manual pages of the command

Process Management

Command Description
ps Display current working processes
pop Display all running processes
kill (pid) Kill the process with given pid
killall (proc) Kill all processes named proc
pkill (pattern) Kill all processes matching the pattern
bg 1.List stopped or background processes 2.Resume a stopped job
fg Bring the recent job to foreground
fg (n) Bring job n to foreground
